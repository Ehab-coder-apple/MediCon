**Steps**:

3. Click "Apply Filters"
4. View all today's records

### Task 2: Check Employee Attendance

**Steps**:
1. Go to Attendance section
2. Select employee from dropdown
3. Set date range
4. Click "Apply Filters"
5. View employee's records

**Time**: 2 minutes

### Task 3: View GPS Coordinates

**Steps**:
1. Go to Attendance section
2. Find record
3. Click "View" button
4. See GPS coordinates:
   - Check-In: Latitude, Longitude
   - Check-Out: Latitude, Longitude
   - Distance from branch

**Time**: 1 minute

**Steps**:

## FILTER QUICK REFERENCE

### Date Range
- **Today**: Select today's date
- **This Week**: Last 7 days
- **This Month**: Last 30 days
- **Custom**: Select start and end dates

### Employee
- **All**: Show all employees
- **Specific**: Select employee name

### Branch
- **All**: Show all branches
- **Specific**: Select branch name

### Status
- **All**: Show all statuses
- **Pending**: Not checked in
- **Checked In**: Currently at work
- **Checked Out**: Left work
- **Incomplete**: Missing check-out

### Geofence
- **All**: Show all records
- **Within**: Inside 300m radius
- **Outside**: Outside 300m radius
