## Detailed Connection Steps

### Step 1: Prepare Your Barcode Scanner

**What You Need:**
- Bluetooth barcode scanner (e.g., Zebra DS3678, Honeywell 1911i, Symbol LS2208)
- Computer with Bluetooth capability
- MediCon web app open in browser

**Check Your Scanner:**
- Ensure battery is charged
- Check if it has Bluetooth (not just USB)
- Find the pairing button (usually on back or side)

### Step 2: Enable Bluetooth on Your Computer

**macOS:**
```
System Preferences ‚Üí Bluetooth ‚Üí Turn On Bluetooth
```

**Windows:**
```
Settings ‚Üí Devices ‚Üí Bluetooth & other devices ‚Üí Turn On Bluetooth
```

**Linux:**
```
Settings ‚Üí Bluetooth ‚Üí Turn On
```

### Step 3: Put Scanner in Pairing Mode

**Most Common Method:**
1. Hold the power button for 3-5 seconds
2. Look for LED indicator (usually blue/red flashing)
3. Scanner is now discoverable

**Check Your Manual For:**
- Exact pairing button location
- How long to hold (usually 3-5 seconds)
- LED indicator colors

### Step 4: Pair the Scanner

**macOS:**
1. Go to System Preferences ‚Üí Bluetooth
2. Click "Pair" next to your scanner name
3. Wait for "Connected" status
4. Done!

**Windows:**
1. Go to Settings ‚Üí Bluetooth & other devices
2. Click "Add Bluetooth or other device"
3. Select "Bluetooth"
4. Click your scanner name
5. Click "Pair"

### Step 5: Test the Connection

**Quick Test:**
1. Open any text editor (Notepad, Word, etc.)
2. Click in the text field
3. Scan a barcode with your scanner
4. Text should appear automatically
5. If it works here, it will work in MediCon!

## Troubleshooting

### Scanner Not Pairing

**Problem:** Scanner doesn't appear in Bluetooth list

**Solutions:**
1. Restart scanner (turn off/on)
2. Restart computer Bluetooth
3. Check battery level
4. Try pairing again
5. Check scanner manual for pairing instructions

### Scanner Paired But Not Working

**Problem:** Scanner is connected but no text appears

**Solutions:**
1. Test in text editor first (Notepad)
2. Check if scanner is in "keyboard mode"
3. Some scanners need mode switching (check manual)
4. Try scanning different barcode format
5. Restart scanner and try again

### Text Appears But Not in MediCon

**Problem:** Scanner works in text editor but not in web app

**Solutions:**
1. Click in the barcode field first
2. Make sure field is focused (cursor visible)
3. Check if JavaScript is enabled in browser
4. Try refreshing the page
5. Check browser console for errors (F12)

### Slow Response

**Problem:** Barcode scans but takes time to appear

**Solutions:**
1. Check internet connection
2. Check if server is responding
3. Try scanning again
4. Restart browser
5. Check MediCon server status

## Scanner Types & Connection Methods

### Bluetooth Scanners (Recommended)
- **Connection:** Wireless via Bluetooth
- **Setup Time:** 5 minutes
- **Range:** 10-30 meters
- **Examples:** Zebra DS3678, Honeywell 1911i
- **Cost:** $100-300

### USB Scanners
- **Connection:** Direct USB cable
- **Setup Time:** 1 minute (plug & play)
- **Range:** Limited by cable
- **Examples:** Symbol LS2208, Honeywell 1900
- **Cost:** $50-150

### Network Scanners
- **Connection:** WiFi/Ethernet
- **Setup Time:** 10 minutes
- **Range:** Entire network
- **Examples:** Zebra DS3678-ER, Honeywell 3320G
- **Cost:** $200-500

## FAQ

**Q: Do I need to install software?**
A: No! Bluetooth scanners work as keyboard input. Just pair and use.

**Q: Can multiple people use same scanner?**
A: Yes, but only one device at a time. Unpair from one, pair with another.

**Q: What if barcode doesn't scan?**
A: Try different angle, clean lens, or check barcode isn't damaged.

**Q: Can I use scanner on mobile app?**
A: Yes! Mobile app also supports barcode scanners via Bluetooth.

**Q: What barcode formats work?**
A: All formats! EAN-13, Code128, UPC, QR codes, etc.
